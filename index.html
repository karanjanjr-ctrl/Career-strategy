How to Update This Site

No new repo needed — ever
1
Go to your GitHub repo
Open github.com → your profile → career-strategy repo.
2
Click index.html → pencil icon
GitHub opens an inline editor. You can edit any text directly here — no code editor needed.
3
Find the data you want to change
Press Ctrl+F (or Cmd+F on Mac) to search. All your firms, tiers, checklist items, and timeline entries live in the data arrays near the top of the <script> section. Search for any text you want to change.
4
Edit and commit
Make your changes. Scroll to the bottom, click "Commit changes", add a short note like "Updated firm list". Site updates in about 60 seconds.

3

4
Commit changes
Scroll down, click "Commit changes". Your live site updates in ~60 seconds. Same URL, no new repo needed.
CHECKLISTS array
Add or remove checklist items. Each group has a name and an items array. Add a new string to items to add a task. Search for "const CHECKLISTS" to find it.
TIERS array
Update comp figures, add new exit paths, change internship targets. Search "const TIERS" to find it.
FIRMS array
Add new firms, update recruiting timelines, change probability ratings as you learn more. Search "const FIRMS".
TIMELINE array
Update milestones as your plan evolves. Add new entries or change dates. Search "const TIMELINE".
SYSTEM_PROMPT (AI Advisor)
The AI advisor's full context about you. Update this as your situation changes — new internships, new goals, plan pivots. Search "const SYSTEM_PROMPT". This is what gives the advisor its memory of who you are.
